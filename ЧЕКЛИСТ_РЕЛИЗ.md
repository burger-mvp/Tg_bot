## Быстрые SQL-запросы для проверки

### Проверить shop_counter
```sql
SELECT nextval('shop_counter');
```

### Посмотреть всех пользователей с shop_name
```sql
SELECT telegram_id, shop_name, role FROM users WHERE shop_name IS NOT NULL;
```

### Найти доверенных продавцов
```sql
SELECT telegram_id, shop_name, phone_number FROM users WHERE role = 'trusted_seller';
```

### Посмотреть отклонённые посты
```sql
SELECT id, author_telegram_id, status, created_at FROM post_queue WHERE status = 'rejected';
```

### Посмотреть последние опубликованные посты
```sql
SELECT 
    pq.id, 
    pq.author_telegram_id, 
    u.shop_name, 
    pq.published_at
FROM post_queue pq
JOIN users u ON u.telegram_id = pq.author_telegram_id
WHERE pq.status = 'published'
ORDER BY pq.published_at DESC
LIMIT 10;
```
